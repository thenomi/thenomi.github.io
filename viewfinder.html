Viewfinder (2023)

UI Art: 2D, 3D, handdrawing

I worked on Sad Owl Studio's Viewfinder as an UI artist in 2023.

I was in charge of designing & creating the title screen, settings menu screen, other menu screens, hand drawn icons, inputs (keys, buttons, mouse), the logo for an in-game virtual pet game Tomodatchi (can you see the Chinese character for 'tomo', 友?), some text boxes, etc.

They asked me to create a menu screen that resembles a DSLR settings screen. Following that style guide, I made photography themed assets such as film roll (which I've also considered animating at one point), vintage photo corners (I've seen them in my grandfather's ancesters family photo albums), memory card, film photographer's work-in-progress, actual viewfinders, etc. and researching them was a fun task to do. (I used to be a photographer and it also felt like that my experience as a photographer finally payed off.)

I also got to work on accessibility options including making sure that the UI would appear fine in various window aspect ratio and that was one of my favourite things about this work.

I also suggested that we should add special cursors for certain actions especially for petting the cat and I'm quite happy with how it turned out.

Viewfinder is currently available on PS4, PS5, and Steam.
